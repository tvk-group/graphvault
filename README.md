# GraphVAULT

GraphVAULT is the semantic knowledge graph and AI memory layer of the SOVRA / Entelekron ecosystem.

## Website

https://www.graphvault.io

## Purpose

GraphVAULT is designed for:

- Semantic entity graphs
- AI memory
- Event-history intelligence
- Provenance mapping
- Document intelligence
- Governance graphs
- Wallet and identity relationship mapping
- ChronoSeal proof integration
- SOVRA agent context

## Deployment

This is a static HTML website. No build step is required.

Vercel settings:

- Framework Preset: Other
- Build Command: empty
- Output Directory: empty
- Install Command: empty
